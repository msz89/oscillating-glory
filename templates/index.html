PASATA

Projection of system availability Time Analysis

  • PASATA retrieves data files from NEMweb for the Medium Term PASA by generator plant DUID.
  • It compares the latest file with the file from the previous period and summarises the changes (PASA-Delta). Postive numbers are an increase in available MW and negative a decrease in available MW
  • The first change for each generator plant is listed below, or if there are no changes, 'No changes' is the status
  • A csv containing the dates and changes can be downloaded using the button
  • By using the endpoint '/mtpasa/[x]' the PASA-Delta can look back 'x' periods. Note that MT PASA is published at 9AM, 12PM, 3PM, 6PM. This should be considered when setting the lookback period.
